CSS

Adding style to elements

Web developers and designers use Cascading Style Sheets to make webpages visually pleasing

Why use CSS?

To describe the look and formatting of a document written in a markup language, eg. hypertext markup language (html)

We can change the colors, sizes, positions, and formatting of any of our HTML elements including the body, paragraphs, links, and images

You can add style directly onto an element

Block and inline elements

<div> and <span> are used to better group your elements and can apply style to their children

  • <div> is a block element. Block elements always start on a new line and fill the width of their space

  • <span> is an inline element and neither starts on a new line nor fills the space
<div> is general used for grouping and containing elements

<span> is often used for applying group styling to text

Organize it better

You can ddd style scripts to your code to better organize your CSS

 This can allow you to add more complexity
It also helps prevent code duplication

The structure

Identifying your element

Instead of grouping by tags, you can decide how you group your elements by adding attributes to your tags - IDs and Classes

ID vs Class

  • Ids are unique
  • Each element can have only one ID
  • Each page can have only one element with that ID
  • Classes are not unique
  • You can use the same class on multiple elements.
  • You can use multiple classes on the same element.
